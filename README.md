# Point Of Sale

> A Quasar project

## Build Setup

``` bash
# clone repository
$ git clone https://github.com/seanstr/pos.git

# change to pos folder
$ cd pos

# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```
